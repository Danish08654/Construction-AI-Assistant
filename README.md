# Construction AI Assistant

The application helps engineers, architects, contractors, and construction professionals quickly retrieve information from building codes
regulations, construction standards, and technical documents using natural language.

---

# Features:

🏗️ Ask construction-related questions in natural language

📚 Retrieval-Augmented Generation 

🔍 Semantic search across construction documents

📖 Building code lookup

🏢 Structural engineering guidance

⚡ Electrical (NEC) reference

🚰 Plumbing code reference

❄️ HVAC design guidance

♿ ADA accessibility requirements

💰 Construction cost estimation

📋 Building permit information

📑 Source citations for every answer

📊 Knowledge Base Explorer

📖 Construction Quick Reference Guide

🎨 Clean and responsive Streamlit interface
