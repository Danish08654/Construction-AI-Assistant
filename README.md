# Construction-AI-Assistant

The assistant helps contractors, engineers, architects, and construction professionals get accurate answers about:

Building codes (IBC, NEC, UPC)

Structural engineering

Permits & inspections

HVAC & plumbing

ADA compliance

Construction materials

Cost estimation

Safety requirements

---

#  Features

🔍 Construction-specific AI assistant

📚 Retrieval-Augmented Generation (RAG)

⚡ Fast semantic search with FAISS

🧠 Groq LLM integration (Llama 3.3 70B)

🏗️ Domain-focused construction knowledge base

📄 Source-aware answers

💬 Streamlit chat interface

⚡ FastAPI backend

🔎 Raw similarity search

📊 Knowledge base analytics
